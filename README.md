## East Agile Claude Code Plugins

Developer workflows for Claude Code.

### Installation

#### 1. Add the marketplace (required)

```bash
claude plugin marketplace add quanhea/claude-code-plugins
```

#### 2. Install individual plugins

```bash
claude plugin install dev@eastagile
# claude plugin install <another_plugin>@eastagile
```
