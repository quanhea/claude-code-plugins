# Changelog

## 1.0.0

- Complete architecture redesign: Skills as Single Source of Truth
- **Dev**: Core developer workflows (TDD, refactor, review, coverage, domain, debug, describe, dive, context, ask)

### Architecture Changes

- Commands and agents are now thin loaders (~10 lines)
- Skills contain all content (~50-80 lines, high-level only)
- Hooks simplified to single high-level reminders
- No over-engineering or tutorial-style content
